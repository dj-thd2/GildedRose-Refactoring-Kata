# GO Starter

- Run:

```shell
make run
```

- OpenAPI Swagger API:
```shell
make run
```

Then go to http://localhost:8080/swagger/index.html

See documentation: [OpenAPI specification (swagger)](docs/swagger.yaml)

- Run unit tests and coverage:

```shell
make unit-test
```

- Run texttest tests:

```shell
make texttest
```
